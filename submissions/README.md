# Submitting your work

1. **Fork** this repository to your own GitHub account (once, before the school).
2. Work through the hands-on notebooks in your fork (Codespaces or Colab).
3. Save your completed notebook as
   `submissions/<your-github-username>/<notebook-name>.ipynb`.
4. Commit and push:
   ```bash
   git add submissions/<your-github-username>/
   git commit -m "Add my solutions for lecture 4"
   git push
   ```
   (In Codespaces you can also use the Source Control panel on the left;
   in Colab use *File ▸ Save a copy in GitHub* and select your fork and the
   `submissions/<your-github-username>/` path.)
5. *Optional:* open a **pull request** from your fork back to this repository
   so we can look at your work together.

**No deadline.** The school days are packed — these exercises are meant to
stay useful as self-paced material. Submissions arriving days or weeks after
the school are welcome and will be reviewed.
